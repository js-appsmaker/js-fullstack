# 2.3 Ligne de commande

Chaque système d'exploitation \(Mac, Linux, Win\) est livré avec une application pour exécuter des commandes \(par exemple Terminal sous Mac et Linux, Command Prompt sous Windows\).

Afin de vous familiariser avec la ligne de commande, parcourez cette ressource :

* Lire ceci : [Ligne de commande](https://tutorial.djangogirls.org/fr/intro_to_command_line/)

### Pratique

Passons en revue quelques commandes de base que nous aurons tendance à utiliser souvent dans ce tutoriel.

1. Ouvrir le Terminal
2. Se déplacer au niveau du répertoire d'accueil

```text
cd ~
```

    3. Créer le dossier ou répertoire  "projects"

```text
mkdir Projects
```

    4.    Aller dans le dossier "Projects"

```text
cd Projects
```

    4. Créer un fichier dans le dossier "Projects"

```text
touch hello.txt
```

    5. Ouvrir le fichier créé dans un éditeur de texte

```text
// Pour Visual Studio Code, dans le Terminal, saisir : 
code hello.txt
```
